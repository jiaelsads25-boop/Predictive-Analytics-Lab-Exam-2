# Predictive-Analytics-Lab-Exam-2
Project Description
This project performs a binary classification task using a dataset with two features (Feature1, Feature2) and a target variable (Yes/No). The target is converted to numerical form (1/0) for modeling.

Exploratory Data Analysis (EDA)
Checked data structure, summary statistics, and missing values
Analyzed class distribution
Visualized data using a scatter plot
Identified and removed an extreme outlier in Feature1 to improve analysis and model performance

Model Development
Used Logistic Regression for classification
Split data into training (80%) and testing (20%) sets
Applied StandardScaler for feature scaling
Trained the model on processed data

Decision Boundary
Generated using meshgrid and contour plotting
Clearly shows separation between the two classes

Model Evaluation
Evaluated using:
Accuracy Score
Confusion Matrix
Classification Report (Precision, Recall, F1-score)

Conclusion
The model performs well on the dataset, and preprocessing (especially outlier removal) improves both visualization and classification accuracy.
